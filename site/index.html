See who any website talks to.

NetLens watches every network request a page makes in the background and breaks it down by category — trackers, ads, APIs, CDNs, fonts — with a privacy grade. No DevTools, no setup.

Turn the raw network firehose into a clear, categorized picture.

Category breakdown

Analytics, ads, social, tag managers, CDNs, fonts, monitoring, payment, support, video — matched to known vendors.

How it works

Install & pin

Add NetLens to Chrome or Edge and pin it to your toolbar.

Browse normally

It captures requests passively in the background on every tab.

Open the popup

See the privacy grade, category bars, and third-party domains for the active tab.

Go in-depth

Open the full report page for charts, the complete domain table, and export.

Install

Chrome Web Store listing is coming soon. Until then, load it unpacked:

  1. Clone or download toolwizhub-netlens.
  2. Open chrome://extensions and enable Developer mode.
  3. Click Load unpacked and select the extension folder.
  4. Open any site, reload it once, then click the NetLens icon.
